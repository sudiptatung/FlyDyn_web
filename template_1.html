FlyDyn

This program simulates how egg number and adult number vary across generations in single, isolated populations of common fruitfly Drosophila melanogaster using an individual-based, discrete generation, stochastic model. Model schematics can be found here. Adjust parameter values and observe corresponding change in egg and adult timeseries.
